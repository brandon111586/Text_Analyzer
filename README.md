# Text Analyzer - 文本資料分析工具
## 專案內容:運用自然語言處理與資料視覺化技術對文本進行線上分析
### 使用技術:
 - Jieba.js進行文本斷詞處理
 - 停用字過濾、詞頻計算
 - Chart.js進行資料視覺化(WordCloud、BarChart)
 - Github Pages 網站建置
